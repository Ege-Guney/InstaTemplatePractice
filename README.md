# Instagram UI Study

A pixel-focused React implementation of a mobile social-media design, built to practice component composition, responsive styling, image-heavy layouts, and client-side routing.

[**Open the live demo →**](https://egeguneyinsta.netlify.app)

## Highlights

- Componentized home feed, stories, posts, navigation, search, and headers.
- React Router navigation between the main feed, search, profile, and fallback views.
- Custom CSS for the compact mobile layout.
- Local image and icon system mirroring a production design handoff.

## Stack

- React 18
- React Router 6
- CSS
- Create React App
- Netlify

## Run locally

```bash
npm install
npm start
```

Then open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
```

## Project note

This is a front-end design study, not an Instagram backend or a production social network. Its focus is visual translation and reusable React composition.
